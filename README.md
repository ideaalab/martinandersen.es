# martinandersen.es

Página personal de Martín Andersen. Estática, hecha a mano: HTML + CSS + JavaScript vanilla, sin frameworks ni dependencias externas.

- **Bilingüe ES/EN** — diccionario en `js/main.js`, detección automática del idioma del navegador, preferencia guardada en `localStorage`.
- **Sección open source dinámica** — carga los repos desde la API pública de GitHub y los agrupa por topics (`ccs-library`, `ha-addon`, `wp-plugin`), con caché de 1 hora. Para que un repo aparezca, basta con ponerle el topic; no hay nada que mantener aquí.
- **Tipografía** — Space Grotesk autoalojada (`fonts/`), sin peticiones a terceros.
- **Animación del hero** — SVG + SMIL puro, respeta `prefers-reduced-motion`.

Publicada con GitHub Pages en [martinandersen.es](https://martinandersen.es).
